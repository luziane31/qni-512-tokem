# Qni-512-token

## 🌐 Visão Geral
O **Qni-512** é um ativo digital e protocolo de tecnologia fundamentado em soberania e inovação. Este repositório serve como o registro oficial e base de código para o ecossistema Qni-512.

## 🚀 Especificações Técnicas
- **Nome:** Qni-512 Token
- **Símbolo:** QNI512
- **Mecanismo:** Proof of Activity (Mineração por Fluxo)
- **Segurança:** Contrato auditado e imutável após deploy.
